# Website-sederhana
Nama: Andrian Maulana Aziz
Ide/Inovasi : Aplikasi Pengingat&Pemantau obat lansia
Temuan Masalah: Banyak lansia yang lupa minum obat atau salah minum obat
Keluarga/Anak kesulitan memantau jadwal konsumsi obat orang tua
Alternatif Solusi :Aplikasi pengingat minum obat dengan notifikasi suara dan visual besar untuk lansia.
Fitur:
Sinkronisasi akun keluarga agar anak bisa memantau.
Peringatan otomatis jika obat belum diminum.
Riwayat kesehatan & rekomendasi dokter online.
Dampak: mengurangi risiko penyakit kronis kambuh karena kelalaian obat.
